Ayuda a debuggear el problema descrito.

## Proceso
1. Entender: ¿Qué debería pasar vs qué pasa?
2. Recopilar: Logs, errores, estado
3. Aislar: ¿Dónde ocurre exactamente?
4. Hipótesis: Causas probables ordenadas
5. Solución: Código o pasos para resolver

Pregunta por mensajes de error o contexto adicional si es necesario.
