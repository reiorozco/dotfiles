Crea un componente siguiendo las convenciones del proyecto.

## Pasos
1. Detecta el framework (React, Vue, Svelte, Angular, etc.)
2. Revisa componentes existentes para seguir el mismo patrón
3. Crea el componente con:
   - Props tipadas
   - Estado local si es necesario
   - Estilos según convención del proyecto

Pregunta el nombre y propósito si no se especifica.
