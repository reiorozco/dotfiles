Crea un endpoint API siguiendo las convenciones del proyecto.

## Pasos
1. Detecta el framework (Express, Fastify, SvelteKit, Next.js, etc.)
2. Revisa endpoints existentes para seguir el mismo patrón
3. Crea el endpoint con:
   - Validación de inputs
   - Manejo de errores
   - Códigos HTTP correctos
   - Tipos de TypeScript (si aplica)

Pregunta qué recurso/entidad maneja si no se especifica.
