Analiza el código y propone refactorizaciones.

## Qué buscar
- Código duplicado
- Funciones largas
- Complejidad innecesaria
- Magic numbers/strings
- Naming mejorable

## Formato
Para cada sugerencia muestra:
1. Problema identificado
2. Código antes
3. Código después
4. Justificación breve

No cambies funcionalidad, solo estructura.
