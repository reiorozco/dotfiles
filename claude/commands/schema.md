Crea o modifica un schema de base de datos.

## Pasos
1. Detecta el ORM/herramienta (Drizzle, Prisma, TypeORM, Sequelize, SQL puro, etc.)
2. Revisa schemas existentes para seguir el mismo patrón
3. Crea el schema con:
   - Primary key
   - Campos con tipos apropiados
   - Relaciones si aplica
   - Índices para campos de búsqueda frecuente
   - Timestamps (createdAt, updatedAt)

Describe la entidad y sus campos, o proporciona el schema actual para modificar.
